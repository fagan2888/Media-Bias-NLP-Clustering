# Revealing the Omitted - An Exploration of Media Bias in the news coverage of Obamacare

## Project Summery: 

Employs Selenium and BeautifulSoup to scrape over 160k articles across over 8k publishers on Obamacare. Uses TF-IDF and LDA to perform topic modeling which revealed whatâ€™s theoretically omitted in a given article and systematically underrepresented at a publisher level.
